# ar-forced-aligner
Forced alignment for dialectal Arabic

This repository contains three forced alignment models. 

Model 1 has probably the best alignment performance, but doesn't have mid vowels.
Model 2 has the worst performance but the most phones.
Model 3 has comparable performance to Model 1, but was trained on a reduced set of consonants.
